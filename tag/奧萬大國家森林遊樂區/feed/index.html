]]> 2019年1月底，安排了一趟暖冬南投三天兩夜小旅行，足跡橫跨清境、合歡山、奧萬大、以及日月潭。旅行交通方式以自助開車為主，其中最令我滿意的景點非 奧萬大 森林遊樂園區莫屬。可能因為暖冬的緣故，部分品種山櫻花提早綻放，不僅可以賞櫻，也同時可賞楓，盡收大自然四季之美。

園區開放時間為每日8:00 am – 5:00 pm

園區導覽地圖

我們來回路線如下：

遊客服務中心 → 櫻花園步道 → 好漢坡 → 沿溪楓林步道 → 賞鳥平台 → 奧萬大吊橋 → 松林區 → 奧萬大吊橋 → 森林公園步道 → 楓林步道 → 木階梯 → 遊客服務中心

來回約2個半小時。建議先在遊客服務中心飲水機補充好水分或帶點小零食在出發。去程走較陡的好漢坡步道上山，回程則改走較平緩的楓林步道與木階梯下山。

整體來說，奧萬大國家森林園區保有非常原始的森林景觀，步道非常寬敞好走，若不是1月底冬天溪流乾凅，不然應該沿溪楓林步道應該會更美。

這次也托暖冬的福，同時看到了櫻花與楓葉，但比較可惜，過吊橋的後的楓林區大部份楓樹都黃了，比較沒有楓紅的感覺。但如果喜歡森林四季景觀的話，這邊真的是一個非常放鬆療癒充滿芬多精的地方。
